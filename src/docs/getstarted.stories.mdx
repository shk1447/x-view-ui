# Get Started

> VASES UI에 대한 설치 및 사용 방법

## Install

```bash
# install locally (recommended)
npm install vases-ui --save
```

## CLI

```bash
# create vases-ui app
npx vases-ui create
```

## Usage

```javascript
import { Units, Theme } from 'vases-ui';

export const Page = () => {
  return <Units.Button>Example</Units.Button>;
};
```
